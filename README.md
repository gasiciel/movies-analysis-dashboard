# Movies Analysis Dashboard

An interactive web dashboard built with Streamlit for exploring and analyzing movie data collected from the Trakt.tv API. The dataset includes 5,000 popular movies from 1960-2024.

**Live demo:** [movies-analysis-dashboard.streamlit.app](https://movies-analysis-dashboard.streamlit.app/)

## Installation

1. Ensure you have Python installed.
2. Install the required dependencies:
```bash
   pip install streamlit pandas matplotlib plotly
```
3. Run the applications locally:
```bash
    streamlit run app.py
```
